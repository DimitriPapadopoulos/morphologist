Dependencies
============

Morphologist-UI depends on a set of other software, mainly developed at NeuroSpin labs, namely the BarinVISA environment: http://brainvisa.info

* Morphologist
* Anatomist
* Axon
* `Capsul <https://github.com/neurospin/capsul>`_

API
===

* :py:mod:`morphologist.core`: core modules.
* :py:mod:`morphologist.gui`: graphical interface modules.
* :py:mod:`morphologist.intra_analysis`: modules related to intra-subject analyses (segmentation, sulci extraction).
